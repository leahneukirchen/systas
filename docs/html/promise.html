Promises

up: Systas Scheme
next: Asynchronous Events
prev: Simple Record Types

A promise represents a lazily-computed value that is cached once it has been computed.

Function promise? 

promise? obj

Return #t if obj is a promise, #f otherwise.



Function make-promise 

(make-promise thunk)

Return a new promise whose forced value is computed by (thunk) . See also xref: delay .
