Multiple Return Values

up: Systas Scheme
next: Low Level Macros
prev: Procedures

Systas permits a procedure to return more than one value using the facilities described here.

Function values 

(values . args)

Return all of the arguments as multiple return values



Function call-with-values 

(call-with-values producer consumer)

Invoke producer , passing all of the values it returns as arguments to consumer .

For example: 

     (call-with-values (lambda () (values 1 2 3))
                       +)
     => 6
