Hash Values

Systas scheme has built-in rules for computing hash values. There are three hash functions, corresponding to the three types of equality (eq? , eqv? , and equal? ).

Function hashq 

(hashq obj n)

Compute a hash value modulo n for obj . eq? objects return equal hash values.



Function hashv 

(hashv obj n)

Compute a hash value modulo n for obj . eqv? objects return equal hash values.



Function hash 

(hash obj n)

Compute a hash value modulo n for obj . equal? objects return equal hash values.



Rationale -- Hash Values

For obvious reasons, the hash value functions mirror the equality functions (see Equality).
