This is a surprisingly handy tool for debugging without needing a complete debugger:

Function peek 

peek . any

Accept any non-0 number of arguments, write them to the current error port as a comment. Return the last argument. 

     (+ 10 (troublesome-fn))
     => (+ 10 (pk 'troublesome-fn-returned (troublesome-fn)))


Function pk 

pk . any

pk is a synonym for peek .
