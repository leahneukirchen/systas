Invoking Systas

up: Systas Scheme
next: An Introduction to Systas Modules

systas [--eval expression]* [file [other arguments]*]

With no argument, systas enters an interactive read-eval-print loop which you can exit by calling (quit) : 

     % systas
     systas> (+ 1 2)
     3
     ...
     systas> (quit)
     %

With arguments, systas interprets each argument in order and then invokes the procedure bound to main . By default, that procedure exits the process with status 0 .

An argument of the form --eval expression simply evaluates expression .

An argument of the form file attempts to load that file.

In either case, if an error occurs, systas exits with non-0 status.

If main is invoked, it is passed a list of strings containing the other arguments .
